# FirstAdminPanel

Ruby gem that adds assets, layouts and generators for admin panel

## Installation

Add this line to your application's Gemfile:

    gem 'first_admin_panel', git: 'git@bitbucket.org:ImmaculatePine/first_admin_panel.git'

And then execute:

    $ bundle
